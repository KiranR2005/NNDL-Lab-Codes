## Neural Networks & Deep Learning Lab (NNDL)

This repository contains practical implementations of core concepts in Neural Networks and Deep Learning using Python. The experiments are designed to provide hands-on experience with real-world AI applications.

---

## Overview

The goal of this repository is to help understand how deep learning models work by applying them to tasks like image classification, object detection, and time series prediction.

---

## Objectives

* Understand fundamentals of deep learning
* Build and train neural network models
* Work with CNN, RNN, and LSTM architectures
* Evaluate model performance using metrics

---

## Experiments

1. Face Recognition System

   * Identifies faces and automates attendance

2. Fraud Detection using ANN

   * Detects fraudulent transactions

3. MNIST Digit Classification

   * Classifies handwritten digits

4. CNN Image Classification

   * Learns image features like edges and shapes

5. RNN Temperature Prediction

   * Predicts future values from sequence data

6. LSTM Anomaly Detection

   * Detects unusual patterns

7. Object Detection (Smart Parking)

   * Detects vehicles using bounding boxes

8. CIFAR-10 Classification

   * Classifies images into multiple categories

9. Pneumonia Detection (Transfer Learning)

   * Uses pre-trained models on medical images

10. YOLO Object Detection

   * Performs real-time object detection

11. Time Series Forecasting (LSTM)

   * Predicts trends using sequential data

## Evaluation Metrics
